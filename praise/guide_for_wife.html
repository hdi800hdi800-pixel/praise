남편이 한 달 동안 터득한
HTML 제작 + 카톡 공유 완전 가이드

✅ STEP 1 — Claude 접속
1
Claude 앱 또는 웹 접속
앱스토어에서 "Claude" 검색 후 설치하거나 아래 주소로 접속
🌐 claude.ai
2
원하는 것을 말로 요청
번역, 요약, 앱 제작, 투표 만들기 등 뭐든지 가능!
✅ STEP 2 — HTML 파일 만들기
💬 이렇게 말하면 돼요
"투표 만들어줘
1. 짜장
2. 짬뽕
3. 볶음밥"
💬 또는 이렇게
"생일 축하 카드 HTML 만들어줘
이름: 홍길동"
💬 사다리타기도 가능
"점심 내기 사다리타기 만들어줘"
💡 핵심 포인트
Claude가 파일을 만들어주면 다운로드 버튼을 눌러 폰에 저장하세요!
✅ STEP 3 — 카톡 링크로 공유
1
Netlify Drop 접속
🌐 app.netlify.com/drop
2
HTML 파일 업로드
"browse files to upload" 탭 → 저장한 HTML 파일 선택
3
"Rename and deploy" 탭
30초 후 링크 자동 생성!
✅ https://랜덤이름.netlify.app
4
링크 복사 → 카톡 붙여넣기 🎉
누구나 클릭하면 바로 실행돼요!
